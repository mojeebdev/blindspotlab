**Blindspot Lab** is a high-performance strategic laboratory focused on identifying structural gaps in Web3 project narratives and roadmaps. Built on 4+ years of deep ecosystem experience (Echelon HQ / Anblabs), this platform leverages an AI-layer to provide instant strategic audits.

## 🛠 The Architecture
- **Framework**: React + Vite + TypeScript
- **Styling**: Tailwind CSS + Shadcn UI
- **Animations**: Framer Motion (Scroll Reveal & Glassmorphism)
- **AI Engine**: Custom Vercel Edge API (V7-Layer Audit)
- **Lead Capture**: Formspree + Zod Validation

## 🧬 Key Features
- **AI Strategy Auditor**: Real-time identification of project "blindspots" using proprietary strategic frameworks.
- **Glassmorphic UI**: High-end agency aesthetic with optimized mobile navigation.
- **Secure Handshake**: Integrated security warnings to prevent impersonation.
- **Single-Page Flow**: Optimized conversion funnel from Value (Audit) to Commitment (Consultation).

## 🚀 Development & Deployment
This project is built for high-scale performance and security.
- **Local Dev**: `npm install` then `npm run dev`
- **Deployment**: Integrated with Vercel for continuous deployment.
- **API**: Connects to the proprietary Blindspot-API layer for generative auditing.
